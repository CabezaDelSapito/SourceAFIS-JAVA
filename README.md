# SourceAFIS JAVA

Geração e Comparação de Templates Biométricos a partir de Imagens PNG usando SourceAFIS em Java

Este projeto utiliza a biblioteca SourceAFIS para gerar templates biométricos a partir de imagens de impressões digitais no formato PNG(amostras coletadas por meio do HID® DigitalPersona® 4500 Fingerprint Reader e DigitalPersona WebSdk JavaScript library). Também permite comparar templates para verificar a correspondência entre diferentes digitais, oferecendo uma solução completa para autenticação biométrica baseada em Java.
